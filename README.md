# ProjectCodeMap: AI開発のためのディレクトリ構造マッピングツール

AIチャットボットにプロジェクトを理解させたいのに、ディレクトリ構造の提示で手こずっていませんか？

`ProjectCodeMap` は、**AI開発（AIDD）を円滑に進めるためのコンテキスト最適化ツール**です。
プロジェクトのディレクトリ構造をスキャンし、AIが最小限のコンテキストでコードベースを理解できるように関数一覧を含めたマッピングを自動生成します。

## 💡 このツールが解決すること

*   **AIへの伝達コストの削減**: 複雑なディレクトリ構造を手動で説明したり、貼り付けたりする手間を解消します。
*   **AIの回答精度の向上**: プロジェクト全体構造と関数定義の概要をコンパクトに伝えることで、AIが正確な箇所を把握しやすくなります。
*   **開発体験（DX）の向上**: ディレクトリ構造を貼り付けるイライラから解放され、本来のコーディング作業に集中できます。

## インストール方法

```bash
pip install .
```

## 使用方法

プロジェクトルートで以下のコマンドを実行するだけです。

```bash
project-code-map
```

実行するとディレクトリ構造と関数一覧が抽出されます。

## ローカルでの実行

```bash
run.bat
```
